## About

Welcome! I'm **Semih**, a software developer based in Istanbul, Türkiye, bridging the gap between low-level systems programming and cutting-edge web development. This repository serves as a central hub for my work, interests, and ongoing projects.

My passion lies in understanding how technology works at every layer — from CAN Bus protocols in automotive systems to React fiber trees in modern web applications. I specialize in **C for embedded and automotive systems** alongside **JavaScript/React for web development**, allowing me to tackle challenges across the full technology stack.

Currently, I'm developing **[EasyGit](https://github.com/semyhist/EasyGit)**, an intuitive Git GUI tool designed to simplify version control workflows. Beyond this, I'm deeply interested in automotive software, embedded systems architecture, and browser internals.

## Getting Started

### Prerequisites

To explore and contribute to projects in this portfolio, you'll need:

- **Git** — For version control and cloning repositories
- **Node.js** (v16+) — For JavaScript/React projects
- **GCC/Clang** — For C compilation and embedded projects
- **A code editor** — VS Code, Vim, or your preferred IDE

### Installation

1. **Clone this repository**
   ```bash
   git clone https://github.com/semyhist/semyhist.git
   cd semyhist
   ```

2. **Navigate to a specific project**
   ```bash
   cd [project-name]
   ```

3. **Install dependencies** (for JavaScript projects)
   ```bash
   npm install
   ```

4. **For C projects**, compile using GCC:
   ```bash
   gcc -o output_file source_file.c
   ```
